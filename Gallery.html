3D Modelling & 2D Work

I’ve used blender to create procedural models and materials, alongside models / materials made with a variety of approaches. I’ve experimented with making materials using photos of real-life materials and handmade touches, and using parameters to try out photo-realistic materials using both Eevee and Cycles. There are some procedural / infinite loops using instances to create multiple copies (enough to loop without noticing) of objects without causing unnecessary render stress.

I’ve experimented with Illustrator to create Vector art in 2D, in addition to creating patterns and using filters such as Gaussian Blur to create glass / fog effects. I made many scrap logos, icons and patterns in addition to using some vector-based work in my personal projects such as in my team’s 2022 Hackathon. Additionally, I have some work with prototyping UI & UX, as seen in the aforementioned hackathon, and my Game Development work.
